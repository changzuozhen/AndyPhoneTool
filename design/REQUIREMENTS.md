# AndyPhoneTool 设计需求

> 使用 [Pencil](https://pencil.dev) 维护 `.pen` 设计稿。请在 Pencil 中新建 `design/AndyPhoneTool.pen` 并打开该文件，Agent 即可通过 Pencil MCP 读写设计。

## 产品定位

跨端手机快捷工具箱（Expo），首页为工具导航，各工具独立全屏页面，便于持续扩展。

## 信息架构

```
AndyPhoneTool
├── 首页（工具导航）
│   ├── 手电筒相机 ✅ v0.1
│   └── 更多工具（占位）
└── 手电筒相机
    ├── 相机预览（全屏）
    ├── 中心对焦框
    ├── 顶部：返回 / 标题 / 缩放比例
    └── 底部：闪光灯状态 + 亮度滑杆
```

## 屏幕 1：首页导航

| 元素 | 说明 |
|------|------|
| 标题 | AndyPhoneTool，28px 粗体 |
| 副标题 | 「手机快捷工具箱 · 一触即达」 |
| 工具卡片 | 深色卡片列表，左图标 + 标题描述 + 右箭头 |
| 可用工具 | 手电筒相机，暖色手电图标 `#FFB347` |
| 占位工具 | 「更多工具 / 即将推出」，降低透明度 |

**画布**：390×844（iPhone 14 逻辑分辨率），背景 `#0F1117`

## 屏幕 2：手电筒相机

| 元素 | 说明 |
|------|------|
| 相机预览 | 全屏后置摄像头，`autofocus=off`（系统连续对焦） |
| 中心对焦框 | 72×72 角标框，居中偏上（42%） |
| 闪光灯 | 进入页面默认 75% 亮度，自动开启 torch |
| 亮度滑杆 | 底部面板，0–100%，低于 5% 关闭闪光灯 |
| 缩放 | 双指捏合，右上角显示当前缩放百分比 |
| 顶部栏 | 半透明黑底，返回按钮 + 标题 + 缩放徽章 |

**画布**：390×844 全屏沉浸式，底部面板圆角 20，背景 `rgba(15,17,23,0.88)`

## 交互说明

1. 首页点击「手电筒相机」→ 全屏进入工具页
2. 进入后自动请求相机权限并开启闪光灯（默认 75%）
3. 拖动亮度滑杆调节闪光灯（iOS 系统级为开关，滑杆映射阈值；Android 后续可接原生亮度 API）
4. 双指捏合缩放，实时显示缩放比例
5. 系统自动连续对焦（中心区域）
